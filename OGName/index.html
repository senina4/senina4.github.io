如何獲得您的 Minecraft bearer token

什麼是Minecraft bearer token?

A Minecraft bearer token 是個JWT token 用於驗證 Minecraft's private APIs.

注意⚠️: Bearer tokens 只能使用 24 小時.

獲得您的bearer token

  • 登入您的 Minecraft account 在 minecraft.net
  • 打開Chrome / Firefox 開發者 (ctrl + shift + i 或 f12) 或 右鍵 然後點 檢查
  • 點擊"控制台"
  • 將此程式碼貼到控制台中：
    console.log(`; ${document.cookie}`.split('; bearer_token=').pop().split(';').shift())
    content_copy
  • 複製控制台中顯示的bearer toeken

您現在擁有了Minecraft帳戶的bearer token.

重要：如果您向某人提供此token，他們可以完全控制您的 Minecraft 帳戶。
